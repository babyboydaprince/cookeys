# Python Chrome cookie grabber and decryptor

Python script which grabs chrome locally stored cookies and drecrypts them for further use.

## 🚀 Run

 ```
python cookeys.py
 ```
### 📋 Prerequisites

-> Python 3


### 🔧 Installation

pip install -r requirements.txt
python cookeys.py


## 🛠️ Made with

-> Python 3.9.13
